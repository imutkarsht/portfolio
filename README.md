# Portfolio


Welcome to my portfolio repository! This portfolio showcases my skills and projects using HTML, CSS, and JavaScript. It's designed to be responsive for all devices and includes a theme switcher button to toggle between day and night themes.

## Sections

1. **Home**: This section provides an introduction to who I am and what I do.

2. **Skills**: Here, you'll find a list of my skills and tech stack.

3. **Projects**: This section showcases some of the projects I've worked on. Each project includes a link to view it.

4. **Contact Me**: If you'd like to get in touch with me, you can find my contact information here.

## Features

- **Theme Switcher**: Toggle between day and night themes for better readability.

- **Dynamic Content**: All content in the sections is generated dynamically using JavaScript.
- **Responsive Design**: The portfolio is designed to be fully responsive, ensuring a great user experience across all devices.

## Usage

To view the portfolio, simply open the [Link](https://imutkarsht-portfolio.vercel.app) in your web browser. From there, you can navigate through the different sections using the menu.
